# process_category_sort

## Purpose
Categorize items from a buyer assortment into e-commerce category rows (CATEGORY_NAME, CATEGORY_ID, ITEM_NAME) for import into the commerce platform.

## How to use
1. Copy the `- Buyer.csv` file generated by `process_buyer_assortments` into this folder
2. Run the script from inside this folder:
   ```
   python process_category_sort.py
   ```
3. When prompted, enter the sport for the assortment (e.g. `Basketball`) or leave blank to skip

## Output
Saved to `~/Downloads/` as:
`Python - [assortment name] - Category.csv`

## Items Not Found
Any item that matched neither a brand nor a product type keyword is listed at the bottom of the output CSV under `# Items Not Found:`. These items need their names reviewed — a keyword may need to be added to the source data.
